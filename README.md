## Project

Users should be able to:

- View responsive website (PC/Mobile)
- See homepage with current top stories news
- In mobile or tablet devices, tap the menu icon to open and close sidebar
- In mobile or tablet view, search bar must be inside sidebar
- In desktop view, click magnifying glass to open search bar
- Search for spesific topic related to keyword entered

## Built with

- Semantic HTML5
- Pure CSS
- Vanilla Javascript
- Javascript DOM
- Bootstrap 5
- HeroIcons
